# KnownIssues.md

버그, 취약한 동작, 패키징 이슈, Unreal 특유의 위험 요소를 기록한다.
문제가 발견되거나 수정될 때마다 이 파일을 업데이트한다.

---

## Template

각 항목은 아래 형식으로 작성한다.

```
## [YYYY-MM-DD] Issue Title

**Status:** Open / Fixed / Workaround Applied

### Symptom
어떤 증상이 나타나는지 기술한다.
에디터에서만 발생하는지, 패키징 빌드에서만 발생하는지 명시한다.

### Cause
원인을 기술한다.

### Fix / Workaround
어떻게 수정했는지 또는 임시 해결책을 기술한다.

### How to Avoid Breaking It Again
재발 방지를 위해 주의해야 할 사항을 기술한다.

### Packaging Build Only
Yes / No
```
